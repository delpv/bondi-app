## 1. Clone the repository
Clone the respository using the following commands:
`git clone https://github.com/delpv/bondi-app.git` 
`cd bondi_repo`

## 2. Install dependencies
Install the dependencing using a package manager:
`npm install` or `yarn install`

## 3. Set up environment variables (see section 4 below)

Check the `.env.example` file which is commited to GitHub, create a copy and rename it to `.env`.
Update the `.env` file with the credentials provided in the email.

# 4. Local development

Run `npm run dev` to start the dev server.
Once `npm run dev` is executed, it will tell you the url and port on which your development server runs. For example you would open `http://localhost:5173` in your browser.
